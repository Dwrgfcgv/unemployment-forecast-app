# Анализ и прогнозирование безработицы

Streamlit-приложение для практической части ВКР. Проект работает с реальными официальными данными по безработице и связанным макроэкономическим факторам, показывает несколько методов анализа связи и строит прогноз временного ряда.

## Что делает приложение

- Загружает официальный набор данных с Росстата и Банка России за период `2015-2024`.
- Позволяет вручную загрузить CSV через интерфейс, если нужно проверить собственный набор данных.
- Не переключается на локальный CSV скрытно: если онлайн-источник недоступен, интерфейс предлагает ручную загрузку CSV.
- Выполняет корреляционный анализ по коэффициентам Pearson и Spearman.
- Сравнивает несколько регрессионных спецификаций и показывает leave-one-out валидацию.
- Показывает стандартизированные коэффициенты и диагностику влияющих лет по Cook's distance.
- Дополняет анализ методом главных компонент и кластеризацией годов.
- Строит прогноз временного ряда с бэктестом и доверительным интервалом.
- Экспортирует проверенный CSV и аналитическую книгу Excel.

  `https://www.rosstat.gov.ru/storage/mediabank/ipc_mes_03-2026.xlsx`

  `https://www.cbr.ru/hd_base/KeyRate/`

## Важная оговорка

Исследование в приложении ведется по одному фиксированному периоду `2015-2024`. Для 2015-2016 годов уровень безработицы сохранен как официальное архивное продолжение ряда Росстата, потому что актуальная федеральная XLSX-таблица начинается с 2017 года. Это позволяет удержать единый десятилетний период для практической части ВКР.

## Запуск

```bash
python -m pip install -r requirements.txt
.venv/bin/streamlit run app.py
```

## Формат CSV для ручной загрузки

Файл должен содержать столбцы:

```text
year, unemployment_rate, average_salary, inflation, real_income_index, gdp_growth, key_rate
```

Шаблон CSV можно скачать в боковой панели приложения.
